# 今順興訂購系統 — 後台設定說明(只需做一次,約 5 分鐘)

後台使用 **Google 試算表 + Apps Script**(完全免費),完成後:

- 每筆訂單自動寫入試算表(「訂單」「明細」「品項統計」三個工作表)
- 自動寄「訂購成功確認信」給客人,同時寄「新訂單通知」給你
- 試算表可隨時 `檔案 → 下載 → Microsoft Excel / CSV` 導出數據

---

## 步驟 1:建立試算表

1. 用你的 Google 帳號開啟 <https://sheet.new>
2. 左上角把試算表命名為:**今順興訂單**

## 步驟 2:貼上後台程式

1. 在試算表上方選單點 **擴充功能 → Apps Script**
2. 把編輯器裡原本的內容全部刪掉
3. 打開本資料夾的 `backend/Code.gs`,**全選複製 → 貼上**
4. 第 16 行 `OWNER_EMAIL` 可改成你要收訂單通知的信箱(目前是 claude04@ydlo.ylc.edu.tw)
5. 按 💾 儲存

## 步驟 3:部署成網頁應用程式

1. 點右上角 **部署 → 新增部署作業**
2. 左邊齒輪 ⚙ 選 **網頁應用程式**
3. 設定:
   - 說明:隨意(例如「訂購系統」)
   - 執行身分:**我**
   - 誰可以存取:**所有人** ← 一定要選這個,客人才送得進訂單
4. 按 **部署** → 第一次會要求授權:選你的帳號 → 進階 → 前往(不安全)→ 允許
5. 複製出現的 **網頁應用程式網址**(https://script.google.com/macros/s/....../exec)

## 步驟 4:把網址填回網站

把複製的網址貼給 Claude 說「幫我把後台網址接上」即可;
或自己到 GitHub 編輯 `config.js`:

1. 開啟 <https://github.com/jackize123/jinshunxing/edit/main/config.js>
2. 把 `window.ORDER_API_URL = "";` 改成 `window.ORDER_API_URL = "你複製的網址";`
3. 按 **Commit changes**,等 1 分鐘後生效

## 步驟 5:測試

1. 開啟訂購網站,隨便下一筆測試訂單(Email 填自己的)
2. 檢查:試算表出現訂單 ✅、兩封信都收到 ✅
3. 測試資料可直接在試算表刪除該列

---

## 日常使用

| 想做的事 | 怎麼做 |
|---|---|
| 看訂單 | 開啟「今順興訂單」試算表 |
| 導出數據 | 試算表 → 檔案 → 下載 → Excel 或 CSV |
| 看各品項總量(備料用) | 「品項統計」工作表,自動即時彙整 |
| 標記已出貨 | 「訂單」工作表最後一欄「狀態」自行改寫 |
| 更換商品照片 | 看網站的 `圖片管理.html` 頁面說明 |

## 修改程式後要更新部署

若日後修改 Code.gs,要點 **部署 → 管理部署作業 → ✏ 編輯 → 版本:新版本 → 部署**,網址不變。
