# 嘉義今順興 — 年菜線上訂購平台

正式網址:**https://jackize123.github.io/jinshunxing/**

| 檔案 | 用途 |
|---|---|
| `index.html` | 訂購網站前台(商品、購物車、冷凍宅配運費、訂購表單) |
| `config.js` | 前台設定:後台 Apps Script 網址、店家信箱 |
| `backend/Code.gs` | 後台程式(貼到 Google Apps Script,見「後台設定說明.md」) |
| `後台設定說明.md` | 後台一次性設定教學(5 分鐘) |
| `圖片管理.html` | 商品圖片專區:每道菜的檔名對照與上傳狀態 |
| `images/products/` | 商品照片資料夾(p01.jpg ~ p20.jpg) |

## 冷凍宅配運費(建置時查詢之公告價)

- 全家 冷凍店到店:$99/箱(取貨付款 $114),限重 10kg、三邊總長 ≦105cm、單邊 ≦45cm
- 7-11 冷凍交貨便:$200/箱(不定期優惠),限重 10kg、長寬高合計 ≦105cm、最長邊 ≦45cm
- 黑貓 低溫宅急便(冷凍):60cm $160/90cm $225/120cm $290,限重 20kg
- 萊爾富:僅常溫店到店,無冷凍服務

運費若有調整,修改 `index.html` 內的 `SHIP` 設定與配送說明表格即可。
